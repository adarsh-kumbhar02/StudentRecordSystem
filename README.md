# Student Record Management System
A web application for managing student records ( ID, Name, Email, Course) 
build with ASP.NET Core MVC, Entity Framework Core, and SQL Server

# Features
 - Add, Edit, Delete, View student records.
 - Identity authentication with role based access:
     - Admin - full CRUD access to all records
     - User - can view all records, but only add/edit/delete their own

# Tech Stack 
- ASP.NET Core MVC (c#)
- Entity Framework Core (Code-First)
- SQL Server
- ASP.NET Core Identity
- jQuery/ AJAX
- Bootstrap
